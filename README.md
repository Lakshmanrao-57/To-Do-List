# Todo List Web Application

# About
This is a simple Todo List web application created using HTML, CSS, and JavaScript.
It helps users to add and manage their daily tasks easily.
This project is made for learning and practice purposes.

Features:-
- Add new tasks
- Delete tasks
- Simple and clean design
- Runs in any web browser

Technologies:-
- HTML
- CSS
- JavaScript

Project Files:-
- index.html
- style.css
- script.js

Purpose:-
This project is created to improve frontend skills and understand basic JavaScript concepts.
It is also useful for resume and internship applications.
